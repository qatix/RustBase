# 所有权
https://www.runoob.com/rust/rust-ownership.html

所有权有以下三条规则：
```aidl
Rust 中的每个值都有一个变量，称为其所有者。
一次只能有一个所有者。
当所有者不在程序运行范围时，该值将被删除。
```
变量范围
```
{
    // 在声明以前，变量 s 无效
    let s = "runoob";
    // 这里是变量 s 的可用范围
}
// 变量范围已经结束，变量 s 无效
```
